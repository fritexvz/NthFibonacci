Nth Fibonacci Number

Nth Fibonacci number calculator

Welcome to Nth Fibonacci number calculator.
It takes input from user to give back the user Nth Fibonacci number with time taken for calculation.
It assumes that the Fibonacci series start with default value which is 1,1,2,3,5,8,13 etc.
For example, if user input 6, which will be the value of N and calculator will return 8 as output which is indeed 6th Fibonacci number starting with default number.
Fibonacci calculator is built with Django, Bootstrap and Jquery frameworks.
